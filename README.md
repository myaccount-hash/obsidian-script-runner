# Script Runner

`.scripts` ディレクトリ配下のスクリプトを実行できるプラグインです。ソースコード中の `PASS_CURRENT_FILE` を `true` にするとスクリプトにファイルを引数として渡します．

Run scripts from the `.scripts` directory in your Obsidian vault. When `PASS_CURRENT_FILE` is set to `true` in the source code, the current file is passed to the script as an argument.
